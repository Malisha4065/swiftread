# **App Name**: SwiftRead

## Core Features:

- PDF Upload: Upload a PDF document for speed reading.
- Word Highlighting: Display words one by one with a highlighted focal point.
- WPM Adjustment: Allow users to adjust words per minute (WPM) during reading.
- Pause/Play: Implement a pause/play button to control reading flow.

## Style Guidelines:

- Background color: Dark charcoal gray (#121212) to minimize distraction and eye strain.
- Primary color: Off-white (#F0F0F0) for text to provide good contrast against the dark background.
- Accent color: Vivid red (#FF4136) to highlight the focal point letter, aiding concentration.
- Font: 'Inter' sans-serif for its clean and legible design, suitable for extended reading.
- Ensure the highlighted word is centered on the screen. Keep static top and bottom bars displaying WPM etc.
- Use a simple, consistent crossfade animation when words switch to reduce motion sickness.
